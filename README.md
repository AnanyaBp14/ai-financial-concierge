# 💰 AI Financial Concierge

A personalized AI-powered financial assistant that helps users make smarter investment decisions through conversation.

---

## 🚀 Overview

AI Financial Concierge is a web application that acts as a personal financial advisor.  
It understands user profiles and provides tailored guidance for investing, saving, and financial planning.

---

## ✨ Features

- 🧠 **User Profiling**
  - Income-based personalization
  - Goals & interests tracking

- 💬 **Conversational AI**
  - Chat-based interaction
  - Context-aware responses

- 📊 **Smart Recommendations**
  - Investment suggestions
  - Risk-aware advice

- 🎨 **Modern UI/UX**
  - Clean chat interface
  - Minimal and calming design

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS

### Backend
- FastAPI (Python)

### AI Logic
- Rule-based + context-aware system

## ⚙️ How to Run

### 1. Backend
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --reload --port 8001
2. Frontend
cd frontend
npm install
npm start
